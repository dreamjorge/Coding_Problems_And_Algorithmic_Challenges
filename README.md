# Coding Problems Collection

![License](https://img.shields.io/badge/license-MIT-blue.svg)

This repository contains solutions to various coding problems and algorithmic challenges, implemented with a focus on educational value and industry best practices.

## Structure

Each problem is contained within its own directory, including:
*   **Source Code:** Implementations ranging from basic to advanced.
*   **Documentation:** Explanations of the approach and design decisions.
*   **Problem Statement:** A `PROBLEM.md` file describing the specific challenge.

## Solved Problems

| Problem | Description | Languages |
| :--- | :--- | :--- |
| [Design Dynamic Array](./design_dynamic_array) | Implementation of a resizable array (Vector/ArrayList) with multiple complexity levels. | C++ |

## Getting Started

1.  Clone the repository.
2.  Navigate to the specific problem directory (e.g., `cd design_dynamic_array`).
3.  Follow the `README.md` instructions within that directory to build and run the solution.
